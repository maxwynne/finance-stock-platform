# Finance Web App

Website application to help your own personal stock portfolio. 

## Tech Stack/Frameworks:

- Python
- HTML 
- Flask 
- CSS

## Additional Features Implemented

- Stocks Purchase and Sell History Table
- Add Cash-to-Wallet functionality
